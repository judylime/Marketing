Notes

Topics are great for ideas and phrases, but what if you have a lot to say?

You can use the Notes window to include a lot of information without having to add more subtopics.

To add notes:

  1. Select Widget A.
  2. On the Topic menu, click Notes .
  3. The Notes window will open from the right.
  4. Type into the Notes window.
  5. To close the Notes window, click Close in the upper right corner.
