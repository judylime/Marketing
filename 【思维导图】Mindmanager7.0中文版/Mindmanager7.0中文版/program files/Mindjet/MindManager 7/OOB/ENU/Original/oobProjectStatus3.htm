Tracking Progress cont'd

You can easily record the progress of these projects with Map Markers from the Task Info pane.

To track progress:

  1. Select Widget A.
  2. Use the Complete drop-down menu in the Task Info pane to select 50% .
    The marker is added to the selected topic.
  3. Select Widget B.
  4. Use the Complete drop-down menu to select 75% .
    The marker is added to the selected topic.
