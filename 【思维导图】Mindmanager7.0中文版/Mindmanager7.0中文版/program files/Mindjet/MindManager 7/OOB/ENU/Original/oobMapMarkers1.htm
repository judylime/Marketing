Map Markers

A key MindManager feature is Map Markers - meaningful visual cues you add to topics.

Since the part shortage is so important, make it more distinct by adding a high priority marker to it.

To add a map marker:

  1. Select Part Shortage.
  2. Click the Map Markers tab on the right side of the application window.
  3. Click the Priority 1 Map Marker . The marker is added to the selected topic.
